# 我的第一个地理空间项目

这是一个基础的地理空间分析项目，用于演示地理数据处理和分析工作流。

## 项目结构
- `data/`: 存储地理空间数据（原始数据和处理后数据）
- `src/`: 包含Python脚本（数据处理、分析功能等）
- `notebooks/`: Jupyter笔记本（用于探索性分析和可视化）

## 依赖库
- GeoPandas
- Fiona
- Matplotlib
- Rasterio（如需处理栅格数据）

## 分析目标
1. 加载并可视化区域矢量数据
2. 进行基本的空间查询和分析
3. 生成简单的地理空间报告
